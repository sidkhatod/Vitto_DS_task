# Credit Default Risk Analysis for Digital Lending

## Overview
This project analyzes credit card default behavior using the UCI Credit Card Default dataset and develops machine learning models to predict future borrower default risk.

The analysis simulates a real-world FinTech credit-risk workflow involving:
- exploratory data analysis,
- feature engineering,
- predictive modeling,
- explainability,
- and business-oriented risk insights.

---

## Dataset
Source:
UCI Machine Learning Repository — Default of Credit Card Clients Dataset

- 30,000 client records
- Taiwan credit card data (2005)
- 25 variables

Target Variable:
- `DEFAULT`
    - 1 → Default
    - 0 → Non-default
